Results of [BHV26]

There exists a classical oracle \( \mathcal{O} \) such that

\[ \mathsf{BQP/qpoly}^{\mathcal{O}} \not\subseteq \mathsf{BQP/poly}^{\mathcal{O}} \]

There exists a classical oracle \( \mathcal{O} \) such that

\[ \mathsf{QMA}^{\mathcal{O}} \not\subseteq \mathsf{QCMA}^{\mathcal{O}} \]

  • No modifications to the oracle
  • Relativization barrier
  • Natural limit

Separation characteristics

  • Capture superpolynomial-amount of information.
    A classical string \( w \in \{0,1\}^{\mathrm{poly}(n)} \) encodes polynomially-many bits, a quantum state \( \lvert \psi \rangle \in \mathbb{C}^{2^{\mathrm{poly}(n)}} \) can be in a superposition over all of them.
  • Modify the state.
    Apply a unitary \( U\) which requires a superposition (i.e. switching measurement with \( U\) should not suffice):
    \( \lvert \psi \rangle \xrightarrow{U} U \lvert \psi \rangle \xrightarrow{\;\text{measure}\;} o_1 \color{red}\neq\color{white} \lvert \psi \rangle \xrightarrow{\;\text{measure}\;} o^\prime_1 \xrightarrow{U} o_1\)
  • Argue via power of copies.
    \(w\) can be copied freely, \(\lvert \psi \rangle \) cannot. A QCMA verifier can reuse same witness even after measurement, QMA cannot.

Building block

There exists a relational problem \( R = \{(x, v): x\in \{0,1\}^n, v\in \{0,1\}^{\textup{poly}(n)}\} \) with the following properties:

  • There exists a polynomial-sized state \(\color{red} \lvert \psi \rangle \) and a machine \(M\) such that for any input \( x \), \(M\) outputs a \( v \) such that \( (x, v) \in R \):
    \( \exists\, \lvert \psi \rangle \;\;\forall x \;:\; M(x, \lvert \psi \rangle) \;\xrightarrow{\text{w.h.p.}}\; v \;\text{ s.t. }\; (x, v) \in R \)
  • There does not exist a polynomial-sized classical string \(\color{red}w\) and machine \(M\) which on input \( x \) produces a valid \( v \):
    \( \nexists\, w \in \{0,1\}^{\mathrm{poly}(n)} \;\;\forall x \;:\; M(x, w) \;\xrightarrow{\text{w.h.p.}}\; v \;\text{ s.t. }\; (x, v) \in R \)

Yamakawa-Zhandry / Code-Intersection Problem

Random functions \( F \)

\( n \) independent random functions \[ f_1,\dots,f_n \;:\; \Sigma \to \{0,1\} \]

Code \( C \)

An error-correcting code \[ C \;\subseteq\; \Sigma^n \]

↓↓

Relation \( R_F \)

Given \( x \in \{0,1\}^n \), find \( c \in C \) such that \( f_i(c_i) = x_i \) for all \( i \),

\[ R_F = \bigl\{(x,\, c) \;:\; c \in C,\; \forall i\; f_i(c_i) = x_i \bigr\} \]

[YZ24, BHV26]  There exists a code \( C \) such that \( R_F \) satisfies the building-block properties.

Quantum Algorithm— Setup

Given state

\[ |\phi_x\rangle \otimes |\psi_C\rangle \;\propto\; \sum_{y:\,F(y)=x} |y\rangle \;\otimes\; \frac{1}{\sqrt{|C|}} \sum_{c\in C} |c\rangle \]

Goal state

\[ |\mathrm{Goal}_x\rangle \;\propto\; \sum_{y:\,F(y)=x,\; y\in C} |y\rangle \]

\[ = \sum_{y \in \Sigma^n} [\mathbf{1}_{F(y)=x}\cdot \mathbf{1}_{y\in C}] |y\rangle \]

\[ = \left(\sum_{y\in \Sigma^n}[\mathbf{1}_{F(y)=x}]\lvert y\rangle \right) \odot \left(\sum_{y\in \Sigma^n} [\mathbf{1}_{y\in C}]\lvert y\rangle\right) \]

\[ = \lvert\phi_x\rangle \odot \lvert\psi_C\rangle \]

Quantum Algorithm — Applying QFT

Given state

\[ QFT|\phi_x\rangle \otimes QFT|\psi_C\rangle \]

\[ \propto \sum_{y} \alpha_y \lvert y\rangle \otimes \frac{1}{\sqrt{|C^\perp|}} \sum_{z\in C^\perp} |z\rangle \]

\[ = \sum_{y\in \Sigma^n,\,z\in C^{\perp}} \alpha_y \lvert y + z\rangle \lvert z\rangle \]

Assuming C is a \( \mathbb{F}_{q}\)-linear code,
\(C^{\perp} = \{c^{\prime}: \forall c\in C, c^{\prime}\cdot c = 0 \} \)

Goal state

\[ QFT\lvert\mathrm{Goal}_x\rangle = \]

\[ \phantom{QFT\lvert\mathrm{Goal}_x\rangle} = QFT\lvert\phi_x\rangle \star QFT\lvert\psi_C\rangle \]

\[ \phantom{QFT\lvert\mathrm{Goal}_x\rangle} \propto \sum_{y\in \Sigma^n,\,z\in C^{\perp}} \alpha_y \lvert y + z\rangle \]

Using the Convolution Theorem,
\[QFT\!\left( \ket{\psi_1} \odot \ket{\psi_2} \right) = QFT(\ket{\psi_1}) \star QFT(\ket{\psi_2}) \]

Quantum Algorithm — Decoding

Let \(r = y+z\) and consider the following algorithm:

\(\displaystyle \lvert y+z\rangle\lvert z\rangle\lvert 0\rangle = \lvert r\rangle\lvert z\rangle\lvert 0\rangle \)

\(\displaystyle \xrightarrow{\textup{Dec}_{C^\perp}} \lvert r\rangle\lvert z\rangle\lvert\textup{Dec}_{C^\perp}(r)\rangle \)

\(\displaystyle \xrightarrow{\;w.h.p.\;} \lvert r\rangle\lvert z\rangle\lvert z\rangle \)

\(\displaystyle \xrightarrow{\;sub\;} \lvert r\rangle\lvert 0\rangle\lvert z\rangle \)

\(\displaystyle \xrightarrow{\textup{Dec}_{C^\perp}^{-1}} \lvert y+z\rangle\lvert 0\rangle\lvert 0\rangle \)

\(\textup{Dec}_{C^\perp}(r)\): Decoding algorithm for \(C^\perp\) which succeeds for all \(z\in C^\perp, y\in\Sigma^n\) such that \[\textup{hw}(z-y)\leq \left(\tfrac{1}{2} + \epsilon\right)N\]

Notice that, \[\Pr_{i\in [n]}[y_i = 0] \approx \frac{1}{2}.\] Therefore, \(\textup{Dec}_{C^\perp}\) succeeds w.h.p.

Quantum Algorithm — Putting it together

\(\displaystyle \text{1. Prepare } |\phi_x\rangle_A \otimes |\psi_C\rangle_B \;\propto\; \sum_{y:\,F(y)=x} |y\rangle \;\otimes\; \frac{1}{\sqrt{|C|}} \sum_{c\in C} |c\rangle \)

\(\displaystyle \text{2. Apply } QFT \text{ to both registers} \)

\(\displaystyle \text{3. Add register } B \text{ into register } A: \quad \sum_{a,z} \alpha_a\, |a+z\rangle_A |z\rangle_B \)

\(\displaystyle \text{4. Uncompute register } B \text{ via } \mathrm{Dec}_{C^\perp}: \quad |a+z\rangle_A |z\rangle_B \mapsto |a+z\rangle_A |0\rangle_B \)

\(\displaystyle \text{5. Apply } QFT^{-1} \text{ to register } A \text{ and measure} \;\xrightarrow{\text{w.h.p.}}\; v \text{ s.t. } (x, v) \in R_F \)

Classical Hardness

Definition: List-Recoverability

A code \( C \subseteq \Sigma^n \) is \( (\ell, L) \)-list-recoverable if for any input lists \( S_1, \dots, S_n \subseteq \Sigma \) with \( |S_i| \leq \ell \) for all \( i \), the number of codewords \( c \in C \) with \( c_i \in S_i \) for all \( i \) is at most \( L \):

\[ \forall S_1,\dots,S_n \subseteq \Sigma,\; |S_i| \leq \ell \;:\quad \bigl|\bigl\{ c \in C \;:\; c_i \in S_i \;\forall i \bigr\}\bigr| \;\leq\; L \]

  • After \(n\cdot l\) queries to \(f_1,\dots f_n\), we may only construct \(L\) codewords \(c\in C\). As long as \(L\ll 2^n\), we cannot output a \((x,c)\in R_F\) for all \(x\in \{0,1\}^n\).
  • [BHV26] There exists a \(\mathbb{F}_q\)-linear code \(C\) with good \(\mathrm{Dec}_{C^\perp}\) which is \((l, 2l)\)-list recoverable for all \(l \leq 2^{n^{1/5}}\).

Witness Separation Problem

Input: \( 1^{\textup{poly}(n)} \) - Global property of \(\mathcal{O}_n \)

Let \( S_1, S_2 \) be a random partition of \( \{0,1\}^n \), and let \( \mathcal{O}_n \) be the oracle:

\( \mathcal{O}_n(x, v) = \begin{cases} 1 & \text{if } (x,v)\in R \text{ and } x \in S_1 \\ \perp & \text{if } (x,v)\in R \text{ and } x \in S_2 \\ \bot & \text{otherwise} \end{cases} \)

YES instance

\( S_1 = \{0,1\}^n \)

NO instance

\( \Pr_{x}[x \in S_1] \leq \tfrac{2}{3} \)

QMA algorithm

Given \( \lvert \psi \rangle \), sample \( x \leftarrow \{0,1\}^n \), run \(v^\prime = M(x, \lvert\psi\rangle)\), accept \( \iff \mathcal{O}_n\!\left(x,\, v^{\prime}\right) = 1 \)

QCMA Lower Bound

  • Successful QCMA algorithm \(A\) with witness \(w \implies\) query \( (x,v)\in R_F\text{ s.t. } x\in S_2\).
  • \(A\) must be good at guessing \(v\in C\) and \((x,v)\in R_F\).
  • For the guesser algorithm, \[\Pr[l \text{ codewords}]\geq \underbrace{2^{-|w|}}_{\color{orange}\text{Correct witness}} \cdot \left(\underbrace{\frac{1}{\mathrm{poly}(n)}}_{\color{orange}\text{Good query}}\right)^l.\]

Guesser Algorithm

Let \(S_1=\emptyset\). Guess a random witness \(w\). Repeat the following \(l\) times:

  1. Run \(A(w)\) with oracle \(O[F,S_1]\).
  2. Choose a random query \(q\in[Q]\), measure the input \((x,v)\).
  3. Add \((x,v)\) to \(O[F,S_1]\).

QCMA Lower Bound

Contradiction

\[s(l) = \text{minimum \# of \textbf{distinct} character outputs in } l \text{ codewords} \]

As \(F\) is random, the probability of guessing all symbols correctly is at most \( 2^{-s(l)}\).

For \((l,O(l))\)-list recoverable code \(C\), \(s(l)=O(n\cdot l)\). Therefore,

\[ \Pr[\text{Success}] \geq 2^{-|w|} \cdot \left(\frac{1}{\mathrm{poly}(n)}\right)^l = \mathrm{poly}(n)^{-l} \color{red}\ll\color{white} 2^{-s(l)} \geq \Pr[\text{Success}] \]

Advice Separation Problem

Input: \( x \in \{0,1\}^n \) - Local property of \(\mathcal{O}_n\)

Let \( S_1, S_2 \) be a random partition of \( \{0,1\}^n \), and let \( \mathcal{O}_n \) be the oracle:

\( \mathcal{O}_n(x, v) = \begin{cases} 1 & \text{if } (x,v)\in R \text{ and } x \in S_1 \\ 2 & \text{if } (x,v)\in R \text{ and } x \in S_2 \\ \bot & \text{otherwise} \end{cases} \)

YES instance

\( x \in S_1 \)

NO instance

\( x \in S_2 \)

BQP/qpoly algorithm

Given \( \lvert \psi \rangle \), run \(v^\prime = M(x, \lvert\psi\rangle)\), accept \( \iff \mathcal{O}_n\!\left(x,\, v^{\prime}\right) = 1 \)

BQP/poly Lower Bound

Argument: Any BQP/poly algorithm \(A\) for an input \(x\) on average must query \((x,v)\in R_F\):

  • (Quantum version of) Yao's box lemma: Given a random function \(G: \{0,1\}^n\rightarrow \{0,1\}\) and \(\mathrm{poly}(n)\) advice, no algorithm can guess \(G(x)\) without querying \(G(x)\).
  • Apply similar Guesser argument to the QCMA lower bound (i.e. measuring the query register and building our own oracle).

Contradicting the hardness of the Code-Intersection Problem.

Open Question: Search-to-decision for QMA

For NP, we know how to perform search-to-decision reductions:

Start with satisfiable
\( \varphi(x_1,\dots,x_n) \)

Query:
is \( \varphi\!\mid_{x_i=0} \) satisfiable?

YES \( \Rightarrow x_i=0 \)
NO \( \Rightarrow x_i=1 \)

Repeat for all bits

[INN+21] There exists a search-to-decision for QCMA.
For QMA we have a quantum oracle impossibility argument.
Can it be improved to a classical oracle argument?
